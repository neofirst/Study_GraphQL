1. PM2

1.1. 설치
yarn add pm2 -g

1.2. 실행
yarn pm2 start ecosystem.config.js

1.3. 실행 PM2 List
yarn pm2 list

1.4. 중지
yarn pm2 kill

1.5. 모니터링
yarn pm2 moit
